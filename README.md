# exploracao-dados-renato-barbosa
Atividade da matéria optativa no curso de Sistemas de Informação da UEA - EST Ciência de Dados

Inicialmente foi selecionado um dataset para a atividade, e foi escolhido os dados de alimentação entregue
para famílias necessitadas no Brasil, visto que é um tópico ainda muito recente e relevante para o nosso contexto.
feito isto foi realizada a criação do Notebook no Google Colab para que seja possível estudar o csv utilizando 
métricas e estatísticas descritivas e geração de gráficos para melhor visualização, com isso podemos tirar insights
importantes sobre os dados e assumir informações sobre o status atual de auxílio à famílias necessitadas no Brasil.
